## 4. 模块依赖关系

```mermaid
graph TD
    Auth["认证模块"] --> Product["商品模块"]
    Auth --> Chat["私信模块"]
    Auth --> Favorite["收藏模块"]
    Auth --> Order["订单模块"]
    Auth --> Profile["个人中心模块"]

    Product --> Category["分类模块"]
    Product --> Chat
    Product --> Favorite
    Product --> Order

    AdminAuth["管理员认证"] --> Audit["商品审核"]
    AdminAuth --> UserMgmt["用户管理"]
    AdminAuth --> CatMgmt["分类管理"]
    AdminAuth --> Dashboard["数据概览"]

    Audit --> Product
    CatMgmt --> Category
    UserMgmt --> Auth
```

**关键依赖说明：**
- 用户端所有模块依赖认证模块（需登录）
- 商品审核模块依赖商品模块（审核的是商品）
- 订单模块依赖商品模块（下单对象是商品）
- 私信模块依赖商品模块（会话围绕商品创建）
- 用户管理模块影响认证模块（禁用=无法登录）
